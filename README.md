# Superstore Data Analysis Project

 Overview
This project explores the **Sample - Superstore dataset** through a structured, step-by-step workflow. The goal is to clean, transform, and analyze the data to uncover meaningful business insights.

 Tasks Completed
1. Data Loading & Exploration – Imported the dataset, inspected structure, and generated summary statistics.  
2. Data Cleaning – Identified and handled missing values, converted columns to correct data types.  
3. Feature Engineering – Extracted time-based features (month, year) and created profit margin.  
4. GroupBy & Aggregation– Summarized sales by category, profit by region, top customers, and monthly trends.  
5. Pivot Tables – Built pivot tables for multi-dimensional analysis (category-region, segment-time).  
6. Splitting & Merging – Split dataset into subsets and merged them back for integrated analysis.  
7. Data Visualization– Created line, bar, and pie charts to visualize trends and distributions.  
8. Business Insights – Identified most profitable category, least performing region, sales-profit patterns, and key trends over time.

Key Insights
- Technology category drives the highest profitability.  
- Certain regions underperform and require strategic review.  
- Sales and profit show a positive correlation, though discounts affect margins.  
- Seasonal trends are visible in monthly sales and profit fluctuations.

 Tools Used
- **Python (Pandas, Matplotlib)** for data manipulation and visualization.  
- **Jupyter/Colab Notebooks** for reproducible analysis and documentation.

Purpose
This project demonstrates a **complete data analysis workflow** — from raw data to actionable insights — suitable for academic review, business reporting, or portfolio presentation.
